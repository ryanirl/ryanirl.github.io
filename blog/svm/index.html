Support Vector Machines

Authors: Ryan Peters*
* Core Contributor; Correspondence to ryanirl@icloud.com;

The Support Vector Machine (SVM) is another form of supervised learning of a binary classifier (see my post on the Perceptron). Specifically, SVM’s are a type of margin maximizers, that along with a kernel (more on this some other time), can become a very powerful tool.

Other models for supervised learning of a binary classifier, such the Rosenblatt Perceptron algorithm, can guarantee convergence, but there is no guarantee on “how good” the hyperplane it will find is. The key insight that led to the derivation of SVMs was the idea that we would not only like our model to converge, but would like our model to give us the “best possible” hyperplane. Although, how does one define the “best possible” hyperplane? Well, SVMs define this as the hyperplane that maximizes the distance between the two binary classes. We will define this distance as the margin. Therefore SVMs try to maximize this margin.

The Margin

Given a dataset X∈RDxNX \in \mathbb{R}^{D x N}X∈RDxN with binary labels y∈{−1,1}y \in \{ -1, 1 \}y∈{−1,1}, weights w∈RDx1w \in \mathbb{R}^{D x 1}w∈RDx1, and a bias b∈Rb \in \mathbb{R}b∈R, we define the margin M\mathcal{M}M as the distance from the hyperplane H\mathcal{H}H (given by wTx+b=0w^Tx+b=0wTx+b=0) to the closest point xi∈Xx_{i} \in Xxi​∈X.

To formalize the definition of the margin M\mathcal{M}M lets first formalize the distance from the hyperplane H\mathcal{H}H to some point xix_{i}xi​. First, notice that if we project xix_{i}xi​ onto H\mathcal{H}H we get another point denoted xpx_{p}xp​ (see Figure 1) such that the distance from xix_{i}xi​ to the hyperplane is simply the distance between the two points. More formally, the distance from H\mathcal{H}H to xix_{i}xi​ is ∣∣e⃗∣∣2||\vec{e}||_{2}∣∣e∣∣2​ where e⃗=xi−xp\vec{e} = x_{i} - x_{p}e=xi​−xp​.

Next, notice that because e⃗\vec{e}e represents the minimum distance between xix_{i}xi​ and H\mathcal{H}H that it will also be orthogonal to H\mathcal{H}H and parallel to www. Therefore we can rewrite e⃗\vec{e}e as e⃗=αw\vec{e} = \alpha we=αw, for some unknown α\alphaα. Additionally, since we know xpx_{p}xp​ lies on the hyperplane, then it must be true that wTxp+b=0w^T x_{p} + b = 0wTxp​+b=0. Using this fact, we can derive the following line of reasoning:

wTxp+b=0wT(xi−e⃗)+b=0wT(xi−αw)+b=0wTxi−αwTw+b=0αwTw=wTxi+bα=wTxi+bwTw\begin{gather*} w^T x_{p} + b = 0 \\[5pt] w^T (x_{i} - \vec{e}) + b = 0 \\[5pt] w^T (x_{i} - \alpha w) + b = 0 \\[5pt] w^T x_{i} - \alpha w^T w + b = 0 \\[5pt] \alpha w^T w = w^T x_{i} + b \\[7pt] \alpha = \frac{w^T x_{i} + b}{w^T w} \\ \end{gather*}wTxp​+b=0wT(xi​−e)+b=0wT(xi​−αw)+b=0wTxi​−αwTw+b=0αwTw=wTxi​+bα=wTwwTxi​+b​​

Now, we can use this definition of α\alphaα to rewrite the length ∣∣e⃗∣∣2||\vec{e}||_{2}∣∣e∣∣2​ as a function of www, bbb, and some xix_{i}xi​:

∣∣e⃗∣∣2=e⃗Te⃗=(αwT)(αw)=∣α∣wTw=∣wTxi+b∣wTwwTw=∣wTxi+b∣wTw=∣wTxi+b∣∣∣w∣∣2\begin{aligned} ||\vec{e}||_{2} &= \sqrt{\vec{e}^T \vec{e}} \\[7pt] &= \sqrt{(\alpha w^T) (\alpha w)} \\[7pt] &= |\alpha| \sqrt{w^T w} \\[7pt] &= \frac{|w^T x_{i} + b|}{w^T w} \sqrt{w^T w} \\[7pt] &= \frac{|w^T x_{i} + b|}{\sqrt{w^T w}} \\[7pt] &= \frac{|w^T x_{i} + b|}{|| w ||_{2}} \\ \end{aligned}∣∣e∣∣2​​=eTe​=(αwT)(αw)​=∣α∣wTw​=wTw∣wTxi​+b∣​wTw​=wTw​∣wTxi​+b∣​=∣∣w∣∣2​∣wTxi​+b∣​​

Then finally, the margin M\mathcal{M}M given some hyperplane H\mathcal{H}H and dataset XXX is given by:

M=min⁡xi∣wTxi+b∣∣∣w∣∣2\mathcal{M} = \min_{x_{i}} \frac{|w^T x_{i} + b|}{|| w ||_{2}}M=xi​min​∣∣w∣∣2​∣wTxi​+b∣​

Primal Form Derivation:

SVMs are a type of “margin maximizers”. That is, we want to maximize the MMM. We can formalize this with the following optimization problem:

max⁡w,bmin⁡xi∣wTxi+b∣∣∣w∣∣2max⁡w,b1∣∣w∣∣2min⁡xi∣wTxi+b∣\begin{aligned} \max_{w, b} \min_{x_{i}} & \frac{|w^T x_{i} + b|}{|| w ||_{2}} \\[7pt] \max_{w, b} \frac{1}{|| w ||_{2}} & \min_{x_{i}} |w^T x_{i} + b| \end{aligned}w,bmax​xi​min​w,bmax​∣∣w∣∣2​1​​∣∣w∣∣2​∣wTxi​+b∣​xi​min​∣wTxi​+b∣​

Notice one last thing, with this definition of the margin we could find some hyperplane that is infinitely far from out data that will satisfy this optimization problem. Therefore we must add an essential constraint that: yi(wTxi+b)≥0      ∀xi∈Xy_{i}(w^Tx_{i} + b) \geq 0 \;\;\; \forall x_{i} \in Xyi​(wTxi​+b)≥0∀xi​∈X. Using this, we once again rewrite the optimization problem as:

max⁡w,b1∣∣w∣∣2min⁡xi∣wTxi+b∣subject to    yi(wTxi+b)≥0      ∀xi∈X\begin{aligned} & \max_{w, b} \frac{1}{|| w ||_{2}} \min_{x_{i}} |w^T x_{i} + b| \\[10pt] \text{subject to} & \;\; y_{i}(w^Tx_{i} + b) \geq 0 \;\;\; \forall x_{i} \in X \end{aligned}subject to​w,bmax​∣∣w∣∣2​1​xi​min​∣wTxi​+b∣yi​(wTxi​+b)≥0∀xi​∈X​

The above optimization problem is rather ugly, the maximization of a minimization with an inequality constraint does not look fun to optimize. Preferably we would like to simplify this by removing the inner minimization. To do so, notice that www is scale invariant. That is, cw  ∀c∈Rcw \; \forall c \in \mathbb{R}cw∀c∈R still defines the same hyperplane. Therefore, we can always find some www such that min⁡xi∣wTxi+b∣=1\min_{x_{i}} | w^Tx_{i} + b | = 1minxi​​∣wTxi​+b∣=1. This constraint allows us to simplify the above objective as:

max⁡w,b1∣∣w∣∣2subject to:  yi(wTxi+b)≥0      ∀xi∈Xmin⁡xi∣wTxi+b∣=1\begin{aligned} & \max_{w, b} \frac{1}{|| w ||_{2}} \\[7pt] \text{subject to:} & \; y_{i}(w^Tx_{i} + b) \geq 0 \;\;\; \forall x_{i} \in X \\[5pt] & \min_{x_{i}} | w^Tx_{i} + b | = 1 \end{aligned}subject to:​w,bmax​∣∣w∣∣2​1​yi​(wTxi​+b)≥0∀xi​∈Xxi​min​∣wTxi​+b∣=1​

Then notice that if min⁡xi∣wTxi+b∣=1\min_{x_{i}} | w^Tx_{i} + b | = 1minxi​​∣wTxi​+b∣=1 then it must be true that ∣wTxi+b∣≥1    ∀xi∈X| w^Tx_{i} + b | \geq 1 \;\; \forall x_{i} \in X∣wTxi​+b∣≥1∀xi​∈X, and we can simplify our constraint even further:

max⁡w,b1∣∣w∣∣2subject to:  yi(wTxi+b)≥1      ∀xi∈X\begin{aligned} & \max_{w, b} \frac{1}{|| w ||_{2}} \\[7pt] \text{subject to:} & \; y_{i}(w^Tx_{i} + b) \geq 1 \;\;\; \forall x_{i} \in X \\[5pt] \end{aligned}subject to:​w,bmax​∣∣w∣∣2​1​yi​(wTxi​+b)≥1∀xi​∈X​

As Kilian Weinberger once said, “maximization is for losers.” So notice that the maximization of the reciprocal of f(x)f(x)f(x) is just the minimization of f(x)f(x)f(x). That is, we can rewrite our formulation as:

min⁡w,b∣∣w∣∣2subject to:  yi(wTxi+b)≥1      ∀xi∈X\begin{aligned} & \min_{w, b} || w ||_{2} \\[7pt] \text{subject to:} & \; y_{i}(w^Tx_{i} + b) \geq 1 \;\;\; \forall x_{i} \in X \\[5pt] \end{aligned}subject to:​w,bmin​∣∣w∣∣2​yi​(wTxi​+b)≥1∀xi​∈X​

One last thing before we are done with the derivation of the SVM primal form. We cannot optimize ∣∣w∣∣2|| w ||_{2}∣∣w∣∣2​ as we need it to be twice differentiable where it’s currently linear. So we write out the final SVM primal form as:

min⁡w,b12∣∣w∣∣22subject to:  yi(wTxi+b)≥1      ∀xi∈X\begin{aligned} & \min_{w, b} \frac{1}{2} || w ||_{2}^2 \\[7pt] \text{subject to:} & \; y_{i}(w^Tx_{i} + b) \geq 1 \;\;\; \forall x_{i} \in X \\[5pt] \end{aligned}subject to:​w,bmin​21​∣∣w∣∣22​yi​(wTxi​+b)≥1∀xi​∈X​

This is the primal form of the SVM problem that we aim to optimize over.

The Lagrangian and the Karush-Kuhn-Tucker (KKT) Conditions

Consider the constrained optimization problem with inequality constraints in the form:

min⁡wf(w)subject to:  gi(w)≥0    ∀i=1,2,...,uhi(w)=0    ∀j=1,2,...,v\begin{aligned} & \min_{w} f(w) \\[5pt] \text{subject to:} & \; g_{i}(w) \geq 0 \;\; \forall i = 1, 2, ..., u \\[7pt] & h_{i}(w) = 0 \;\; \forall j = 1, 2, ..., v \\[7pt] \end{aligned}subject to:​wmin​f(w)gi​(w)≥0∀i=1,2,...,uhi​(w)=0∀j=1,2,...,v​

Such that f(w)f(w)f(w) and gi(w)g_{i}(w)gi​(w) are convex, and hi(w)h_i(w)hi​(w) is affine. We call this form the primal optimization problem where the number of inequalities is uuu and the number of equalities is vvv. Furthermore, we define the generalized Lagrangian of this form to be:

L(w,α,β)=f(w)+∑i=1uαigi(w)+∑j=1vβjhj(w)\begin{aligned} \mathcal{L}(w, \alpha, \beta) = f(w) + \sum_{i = 1}^{u} \alpha_{i} g_{i}(w) + \sum_{j = 1}^{v} \beta_{j} h_{j}(w) \\ \end{aligned}L(w,α,β)=f(w)+i=1∑u​αi​gi​(w)+j=1∑v​βj​hj​(w)​

Such that α\alphaα and β\betaβ are the Lagrange multipliers. We would like to know when the generalized Lagrangian is equal to the primal form. To do this first consider two quantities. The first quantity:

P(w)=max⁡α,β:α≥0L(w,α,β)\begin{aligned} \mathcal{P}(w) & = \max_{\alpha, \beta : \alpha \geq 0} \mathcal{L}(w, \alpha, \beta) \\[7pt] \end{aligned}P(w)​=α,β:α≥0max​L(w,α,β)​

Given this quantity we can consider two cases: The first case is when www takes on a value that satisfies the primal constraints. In this case we can see that P(w)=f(w)\mathcal{P}(w) = f(w)P(w)=f(w). In the other case, the case that www violates the primal constraints, we will have P(w)=∞\mathcal{P}(w) = \inftyP(w)=∞. To formalize this we get:

P(w)={f(w)  if w satisfies the primal constraints.∞  if w does NOT satisfy the primal constraints.\mathcal{P}(w) = \begin{cases} f(w) \; \text{if $w$ satisfies the primal constraints.} \\[7pt] \infty \; \text{if $w$ does NOT satisfy the primal constraints.} \\[7pt] \end{cases}P(w)=⎩⎨⎧​f(w)if w satisfies the primal constraints.∞if w does NOT satisfy the primal constraints.​

Thus this form gives the same values of the primal form when www satisfies the primal constraint. Therefore, we can write an equivalent statement to the primal form as the following:

min⁡wP(w)=min⁡wmax⁡α,β:α≥0L(w,α,β)\begin{aligned} \min_{w} \mathcal{P}(w) & = \min_{w} \max_{\alpha, \beta : \alpha \geq 0} \mathcal{L}(w, \alpha, \beta) \\[7pt] \end{aligned}wmin​P(w)​=wmin​α,β:α≥0max​L(w,α,β)​

Now lets consider the second quantity:

D(w)=min⁡wL(w,α,β)\begin{aligned} \mathcal{D}(w) & = \min_{w} \mathcal{L}(w, \alpha, \beta) \\[7pt] \end{aligned}D(w)​=wmin​L(w,α,β)​

Then we define the dual of this form as follows:

max⁡α,β:α≥0D(w)=max⁡α,β:α≥0min⁡wL(w,α,β)\begin{aligned} \max_{\alpha, \beta : \alpha \geq 0} \mathcal{D}(w) & = \max_{\alpha, \beta : \alpha \geq 0} \min_{w} \mathcal{L}(w, \alpha, \beta) \\[7pt] \end{aligned}α,β:α≥0max​D(w)​=α,β:α≥0max​wmin​L(w,α,β)​

Now lets compare these two quantities: Let P^\hat{P}P^ be the optimal value of min⁡wP(w)\min_{w} \mathcal{P}(w)minw​P(w) and D^\hat{D}D^ be the optimal value of maxα,βD(w)max_{\alpha, \beta} \mathcal{D}(w)maxα,β​D(w). Then if follows that because min⁡max⁡≥max⁡min⁡\min \max \geq \max \minminmax≥maxmin that:

min⁡wP(w)≥max⁡α,β:α≥0D(w)min⁡wmax⁡α,β:α≥0L(w,α,β)≥max⁡α,β:α≥0min⁡wL(w,α,β)P^≥D^\begin{aligned} \min_{w} \mathcal{P}(w) & \geq \max_{\alpha, \beta : \alpha \geq 0} \mathcal{D}(w) \\[7pt] \min_{w} \max_{\alpha, \beta : \alpha \geq 0} \mathcal{L}(w, \alpha, \beta) & \geq \max_{\alpha, \beta : \alpha \geq 0} \min_{w} \mathcal{L}(w, \alpha, \beta) \\[7pt] \hat{P} & \geq \hat{D} \\[7pt] \end{aligned}wmin​P(w)wmin​α,β:α≥0max​L(w,α,β)P^​≥α,β:α≥0max​D(w)≥α,β:α≥0max​wmin​L(w,α,β)≥D^​

This inequality P^≥D^\hat{P} \geq \hat{D}P^≥D^ is called weak duality where the difference P^−D^\hat{P} - \hat{D}P^−D^ is called the duality gap. This inequality is always true even under non-convex conditions. Though we would like know under what conditions strong duality (P^=D^\hat{P} = \hat{D}P^=D^) holds, this happens exactly when the duality gap is 0. By Slater’s theorem (I will not be proving it here) we can show that strong duality holds in the case that:

  1. f(x)f(x)f(x) and gi(x)g_i(x)gi​(x) is convex. hj(x)h_j(x)hj​(x) is affine.
  2. ∃w\exists w∃w such that gi(w)<0g_i(w) < 0gi​(w)<0.

For the hard margin SVM, there is no equality constraint hj(x)h_j(x)hj​(x). We know that f(x)f(x)f(x) is convex and gi(x)g_i(x)gi​(x) is affine and therefore convex. Therefore, Slater’s conditions hold under the assumption that ∃w\exists w∃w such that gi(w)<0g_i(w) < 0gi​(w)<0, which is true if there exists some hyperplane that separates our data (as we will let gi(w)=−yi(wTxi+b)+1≤0g_i(w) = -y_i(w^Tx_i + b) + 1 \leq 0gi​(w)=−yi​(wTxi​+b)+1≤0).

Under these conditions we will have some optimal values w^\hat{w}w^, α^\hat{\alpha}α^, and β^\hat{\beta}β^​ that additionally satisfy the Karush-Kuhn-Tucker (KKT) conditions, which are as follows:

  1. ∂∂wiL(w,α,β)=0  ∀i=1,2,...,n\frac{\partial}{\partial w_{i}} \mathcal{L}(w, \alpha, \beta) = 0 \; \forall i = 1, 2, ..., n∂wi​∂​L(w,α,β)=0∀i=1,2,...,n
  2. ∂∂βiL(w,α,β)=0  ∀i=1,2,...,v\frac{\partial}{\partial \beta_{i}} \mathcal{L}(w, \alpha, \beta) = 0 \; \forall i = 1, 2, ..., v∂βi​∂​L(w,α,β)=0∀i=1,2,...,v
  3. αi(gi(w))=0  ∀i=1,2,...,u\alpha_{i}(g_{i}(w)) = 0 \; \forall i = 1, 2, ..., uαi​(gi​(w))=0∀i=1,2,...,u
  4. gi(w)≤0  ∀i=1,2,...,ug_{i}(w) \leq 0 \; \forall i = 1, 2, ..., ugi​(w)≤0∀i=1,2,...,u
  5. αi≥0  ∀i=1,2,...,u\alpha_i \geq 0 \; \forall i = 1, 2, ..., uαi​≥0∀i=1,2,...,u

Dual Form Derivation:

The SVM primal form is a constrained optimization problem and can be solved using Lagrangian multipliers. Taking into account the above notes we let gi(w)=−[yi(wTxi+b)−1]≤0g_i(w) = -[y_i(w^Tx_i + b) - 1] \leq 0gi​(w)=−[yi​(wTxi​+b)−1]≤0. Notice we do not have any equality constraints, then the Lagrangian of the SVM primal form is the following:

L(w,b,α)=12∣∣w∣∣22−∑i=1nαi[yi(wTxi+b)−1]=12wTw−∑i=1n[αiyiwTxi+αiyib−αi]=12wTw−wT∑i=1nαiyixi−b∑i=1nαiyi+∑i=1nαi\begin{aligned} \mathcal{L}(w, b, \alpha) &= \frac{1}{2} || w ||_{2}^2 - \sum_{i = 1}^{n} \alpha_{i} [y_{i}(w^Tx_{i} + b) - 1] \\[7pt] &= \frac{1}{2} w^T w - \sum_{i = 1}^{n} [\alpha_{i} y_{i} w^Tx_{i} + \alpha_{i} y_{i} b - \alpha_{i}] \\[7pt] &= \frac{1}{2} w^T w - w^T \sum_{i = 1}^{n} \alpha_{i} y_{i} x_{i} - b \sum_{i = 1}^{n}\alpha_{i} y_{i} + \sum_{i = 1}^{n}\alpha_{i} \\[7pt] \end{aligned}L(w,b,α)​=21​∣∣w∣∣22​−i=1∑n​αi​[yi​(wTxi​+b)−1]=21​wTw−i=1∑n​[αi​yi​wTxi​+αi​yi​b−αi​]=21​wTw−wTi=1∑n​αi​yi​xi​−bi=1∑n​αi​yi​+i=1∑n​αi​​

Deriving the Weights and Bias from the Dual Form

We already have a known solution for the weights that satisfies the KKT condition #1. That being: w=∑i=1nαiyixiw = \sum_{i = 1}^{n} \alpha_{i} y_{i} x_{i}w=∑i=1n​αi​yi​xi​. Taking into account the KKT condition #3, notice that αi(yi(wTxi+b)−1)=0\alpha_{i}(y_{i}(w^Tx_{i} + b) - 1) = 0αi​(yi​(wTxi​+b)−1)=0 where αi≥0\alpha_i \geq 0αi​≥0. This leaves us with two main conditions, either αi\alpha_{i}αi​ is 0 and we are not dealing with a support vector, or yi(wTxi+b)=1y_{i}(w^Tx_{i} + b) = 1yi​(wTxi​+b)=1 in which case we are dealing with a support vector. We can ignore the case in which αi=0\alpha_{i} = 0αi​=0 and deal solely with the case regarding the support vector. Simplifying the support vector case gives the following:

yi(wTxi+b)=1wTxi+b=1yib=1yi−wTxib=yi−wTxi\begin{aligned} y_{i}(w^Tx_{i} + b) = 1 \\ w^Tx_{i} + b = \frac{1}{y_{i}} \\ b = \frac{1}{y_{i}} - w^Tx_{i} \\ b = y_{i} - w^Tx_{i} \\ \end{aligned}yi​(wTxi​+b)=1wTxi​+b=yi​1​b=yi​1​−wTxi​b=yi​−wTxi​​

The equation above gives the bias for some support vector where αi≥0\alpha_{i} \geq 0αi​≥0. To find the bias of the hyperplane, we can just take the mean average over each support vector bias. Resulting in:

bH=[yi−wTxi]+[yj−wTxj]2=−wTxi+wTxj2subject to:  yi=−1∧αi≥0  yj=1∧αj≥0\begin{aligned} b_{\mathcal{H}} & = \frac{[y_{i} - w^Tx_{i}] + [y_{j} - w^Tx_{j}]}{2} \\[7pt] & = -\frac{w^Tx_{i} + w^Tx_{j}}{2} \\[7pt] \text{subject to:} \; & y_{i} = -1 \land \alpha_{i} \geq 0 \\[5pt] \; & y_{j} = 1 \land \alpha_{j} \geq 0 \\[5pt] \end{aligned}bH​subject to:​=2[yi​−wTxi​]+[yj​−wTxj​]​=−2wTxi​+wTxj​​yi​=−1∧αi​≥0yj​=1∧αj​≥0​

In most cases this will just equal the average over all support vector biases, though this will not always be true. In fact, I have seen multiple sources say that this is true and that the bias is equal to the mean over all support vectors biases, including myself at one point. Though I realized that if you have two points on one of your support vectors then taking the mean over all biases would skew your bias towards that support vector. So, you must be careful to only take the mean average over two bias terms (one on each support vector).

Python Implementation and CVXOPT

CVXOPT is a python library for convex optimization. It uses quadratic programming to solve the convex optimization problem of the following form:

min⁡12xTPx+qTx subject to: Gx≤hAx=b\begin{array}{ll} & \min \frac{1}{2} x^{T} P x+q^{T} x \\ \text { subject to: } & G x \leq h \\ & A x=b \end{array} subject to: ​min21​xTPx+qTxGx≤hAx=b​

Conclusion

In this post, we introduced the Support Vector Machine (SVM), a model for supervised learning of a binary classifier. We explored the mathematical foundations of SVMs, starting from the concept of margin maximization, and then derived both the primal and dual optimization forms. We also introduced and discussed the Karush-Kuhn-Tucker (KKT) conditions, and their role in solving the SVM optimization problem. Finally, we showed a practical implementation of an SVM in Python, using the CVXOPT library, and compared our implementation with scikit-learn’s.
